Инструкция по работе с GIT

Инструкция написана на языке разметки текстов MarkDown (сокр. MD). Язык MD используется для создания красиво оформленных текстовых документов, руководств пользователя, файлов README и т.д.

Документы, написанные с использованием синтаксиса MD, представляют собой обычные текстовые файлы TXT. Их можно открыть на любой платформе и в любом редакторе. Этим они выгодно отличаются от файлов, созданных в текстовых процессорах.
